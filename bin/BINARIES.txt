# Quad Extraction via pyqex

Quad mesh extraction is now handled by the `pyqex` package, which provides
Python bindings for libQEx.

## Installation

Run the install script:

```bash
python scripts/install_pyqex.py
```

Or install manually from: https://github.com/mfagerlund/pyqex/releases

## Usage

The `libqex_wrapper` module automatically uses pyqex:

```python
from rectangular_surface_parameterization.utils.libqex_wrapper import extract_quads

quad_verts, quad_faces, tri_faces = extract_quads(vertices, triangles, uvs)
```
